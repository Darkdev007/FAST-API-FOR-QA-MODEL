# FAST-API-FOR-QA-MODEL
This covers creating an API endpoint with a finetuned BERT model from the huggingface library

A FastAPI project for a Question Answering language model allows users to send a context and a question and receeive an answer generated by a fine-tuned Hugging Face model.

## Features
- Uses a BERT base model fintuned on SQuAD dataset
- Deployable via **FastAPI** for local or cloud hosting.
- Supports sending context and question via POST request to get answers.

## Requirements
- Python 3.8 - 3.12
- FastAPI
- Uvicorn
- Transformers
- PyTorch
- python-dotenv
- requests
